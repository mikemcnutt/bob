Color theory, effective design and design consistency are some of the topics covered in this module. This module covers Internet Protocols, web browsers, the use of graphics and typography, accessibility basic HTML elements. This module will deal with website file management, determining the elements to use when setting the structure of a webpage, adding static content to a webpage template, the use of heading elements, the benefits of validating web documents and creating a home page from an HTML template. This module deals with the use of images throughout a website to enhance visual appeal and provide visitors with additional information about a product or service. We will also cover types of hyperlinks such as relative links, absolute links, bookmark links, email links, and telephone links. We will also work with the different types of lists available for use in an HTML document. This module will cover the importance of separating design from content. We will explore the use of Cascading Style Sheets (CSS), the difference between inline, embedded, and external styles and their order of precedence. We will also learn the difference between a selector, property, and value. Prior to the era of mobile devices, websites were almost exclusively accessed by desktop computers. However, the use of mobile devices to connect to the Internet has been rising since 2010. In this module we discuss the web development approach called responsive design. We will learn about the design principles of a mobile website, the use of viewports and mobile friendly navigation. In this module we will expand on the use of viewports and learn about creating viewports for tablet versions of websites. We will explore the design principles of a tablet website, create a media query to target tablet viewports and also create media queries to target larger desktop viewports.
