Module 9 - Creating Tables and Forms

Module Overview

Content you can organize into categories is often best presented in a table format. A table presents related information in rows and columns, and is especially useful when comparing types of data or listing topics and details. Tables can use column headings or row headings to identify categories or topics. Tables are also helpful when you need to provide a lot of content in a compact form.

Many businesses use forms to collect information about their customers. Common information to collect includes a customer's first and last names, address, email address, and phone number. Websites provide forms so visitors can create an account, register for an event, or make a purchase, for example.

In this chapter, you will learn how to use HTML to create a table and a form on a webpage. After creating the table using several HTML elements, you format the table using CSS. To create the form, you will include several controls on a webpage, including check boxes, a drop-down menu, and text boxes. You will also learn how to add a Submit button that customers can use to submit the completed form. Finally, you will learn to style the form elements using CSS.

Why It Matters

Tables are indispensable for organizing and presenting categorized content efficiently. Whether comparing data or listing topics and details, tables provide a structured format that enhances readability and comprehension. They're particularly useful when you need to present a lot of information compactly, making them a valuable tool for businesses and websites alike.

Forms, on the other hand, are essential for collecting information from website visitors. Businesses commonly use forms to gather customer details such as names, addresses, email addresses, and phone numbers. Whether it's for account creation, event registration, or making purchases, forms facilitate crucial interactions between businesses and their customers.
