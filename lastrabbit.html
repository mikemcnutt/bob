Color theory, effective design and design consistency are some of the topics covered in this module. This module covers Internet Protocols, web browsers, the use of graphics and typography, accessibility basic HTML elements. This module will deal with website file management, determining the elements to use when setting the structure of a webpage, adding static content to a webpage template, the use of heading elements, the benefits of validating web documents and creating a home page from an HTML template. This module deals with the use of images throughout a website to enhance visual appeal and provide visitors with additional information about a product or service. We will also cover types of hyperlinks such as relative links, absolute links, bookmark links, email links, and telephone links. We will also work with the different types of lists available for use in an HTML document. This module will cover the importance of separating design from content. We will explore the use of Cascading Style Sheets (CSS), the difference between inline, embedded, and external styles and their order of precedence. We will also learn the difference between a selector, property, and value. Prior to the era of mobile devices, websites were almost exclusively accessed by desktop computers. However, the use of mobile devices to connect to the Internet has been rising since 2010. In this module we discuss the web development approach called responsive design. We will learn about the design principles of a mobile website, the use of viewports and mobile friendly navigation. In this module we will expand on the use of viewports and learn about creating viewports for tablet versions of websites. We will explore the design principles of a tablet website, create a media query to target tablet viewports and also create media queries to target larger desktop viewports. As you have learned in previous modules, HTML 5 provides tools for improving webpage design, including semantic elements for specific types of content. You have already integrated four semantic elements in a website: header, nav, main, and footer. In this module, you will discover other HTML 5 semantic elements and learn how to integrate them into webpages. You will also learn how to use other CSS properties to further improve your current design for mobile, tablet, and desktop viewports. In this module, you will learn how to use HTML to create a table and a form on a webpage. After creating the table using several HTML elements, you format the table using CSS. To create the form, you will include several controls on a webpage, including check boxes, a drop-down menu, and text boxes. You will also learn how to add a Submit button that customers can use to submit the completed form. Finally, you will learn to style the form elements using CSS. In this module, you will learn how to use HTML 5 to embed audio and video on an HTML webpage. You will learn how to use the audio element and its attributes to add audio to a webpage. Similarly, you will learn how to use the video element and its attributes to add a video to webpage. You will also learn about the various audio and video file formats and which formats to use for the web. Finally, you will learn how to make videos accessible. In this module, you will explore how to integrate animation on a webpage using CSS and JavaScript. You will learn how to use the CSS transform property to make elements move on a webpage. You will also use CSS to create an animation using keyframes. Next, you will learn how to create an external JavaScript file and write several functions. Finally, you will learn how to reference the external JavaScript file within an HTML webpage. In this module, you will learn how to publish and promote your website. You will learn about linking your social media pages to your website. You will improve a website for search engine optimization and use the meta tag to add a description to a webpage. You will also learn how to connect to a remote server, publish your website, and discover the many ways to promote and market your website. You will create a Skip to Content link to make the website accessible to visually impaired users and minify your CSS file. You also will learn about the development life cycle of a website, and about the project management process, including content and design updates. Finally, you will learn more about e-commerce websites. This module provides an introduction to Bootstrap, a popular front-end web framework used to create responsive webpages. You will learn how to create a webpage using Bootstrap’s starter template, and then integrate and design a responsive navigation system using the Bootstrap framework. You will learn how to create a hero feature using the Bootstrap framework, to use the Bootstrap grid system to lay out webpage content, and to style text, images, and a table using the Bootstrap framework.
