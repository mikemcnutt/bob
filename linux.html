MODULE 3: Working in the Shell & Managing Files

In the previous chapter, you learned about navigating the Linux filesystem as well as viewing and editing files. This chapter focuses on the organization of files on the Linux filesystem as well as their linking and security. First, you explore standard Linux directories using the Filesystem Hierarchy Standard. Next, you explore common commands used to manage files and directories as well as learn methods that are used to find files and directories on the filesystems. Finally, you learn about file and directory linking, permissions, special permissions, and attributes.

Why It Matters

Why These Concepts Matter

Building upon the foundational knowledge acquired in the previous chapter, this chapter delves deeper into the organization, management, and security of files within the Linux filesystem. You will gain a comprehensive understanding of file organization using the Filesystem Hierarchy Standard, exploring standard Linux directories and their significance.

Through the exploration of common commands for managing files and directories, you will acquire practical skills for efficient file manipulation and navigation. Additionally, methods for locating files and directories on filesystems will be introduced, enhancing your ability to retrieve and manage data effectively.

Furthermore, you will delve into file and directory linking, permissions, special permissions, and attributes, crucial aspects of file security and access control within Linux environments. By mastering these concepts and techniques, you will strengthen your ability to organize, manage, and secure files effectively, essential skills for navigating and operating within Linux computing environments.

Demonstrate Competency

  • Modify the default access control list (ACL)
  • View and set filesystem attributes
