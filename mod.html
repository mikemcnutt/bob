Module Overview

In this module, you'll examine the different tools that can be used to monitor and manage Windows Server 2022 in more depth, including Server Manager, the Windows Admin Center, Control Panel, and Device Manager. You'll also learn how to check for and repair system files, as well as edit the Windows Registry. At the end of the module, you'll learn how to work within Windows PowerShell, as well as how to create and modify PowerShell scripts to automate system configuration tasks.

Why Does It Matter?

This module serves as a pivotal element in the skill set of any IT professional. The tools and techniques covered are not just theoretical knowledge but provide students with hands-on experience in managing, troubleshooting, and automating Windows Server 2019. The skills acquired are directly applicable in diverse professional settings, making this module an indispensable part of their academic journey.

Demonstrate Competencies

After completing this module, you will be able to:

  • Use Server Manager to monitor and manage Windows Server systems
  • Use Server Manager to monitor and manage Windows Server systems
  • Configure server hardware devices
  • Use the System File Checker and Sigverif to verify system files
  • Configure key Windows Server components within Control Panel and Device Manager
  • Explain the purpose and configuration of the Windows Registry
  • Identify the components, features, and usage of Windows PowerShell
  • Identify the components, features, and usage of Windows PowerShell
  • Create PowerShell scripts for systems administration
