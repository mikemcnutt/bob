Module Overview

Today, millions of people worldwide have access to the Internet, the world's largest network. Billions of webpages provide information on any subject you can imagine. People use the Internet to search, communicate, and be entertained. Businesses rely on it to sell products and services. HTML and CSS make this possible.

HTML 5.3 is the most recent version of HTML and is called HTML 5. You'll learn the basics of the Internet, how it works, website types, how to plan a website for an audience, and how to create a basic webpage using a text editor.
